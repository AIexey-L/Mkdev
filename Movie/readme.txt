Задачи

Установить Ruby современной версии и текстовый редактор с подсветкой синтаксиса
Написать программу movies.rb, которая будет запускаться с одним аргументом, вот так: ruby movies.rb Titanic (здесь "Titanic" это аргумент скрипта)
...и будет для аргумента "Matrix" выводить "Matrix is a good movie", для аргумента "Titanic" — "Titanic is a bad movie", а для всех остальных — "Haven't seen (название фильма) yet"
Хранить названия хорошего и плохого фильмов (ты можешь их заменить) в переменных
Выложить программу на http://gist.github.com (не забудь установить правильный тип файла! он определяется расширением .rb)
сложнее Попробуем массивы? Допустим, у нас несколько любимых фильмов. Давай хранить их список в массиве, и немного усложним проверку — хороший ли этот фильм (входит ли он в наш список?).
Подсказка: аргументы, переданные скрипту, можно получить использовав ARGV.
