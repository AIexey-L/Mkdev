Задачи

Хранить каждый фильм в словаре (Hash) — ключами такого словаря будут, например, :link, :title, и т.д. — по смыслу полей в файле
Нужно вывести:
5 самых длинных фильмов;
10 комедий (первые по дате выхода);
список всех режиссёров по алфавиту (без повторов!) — кстати, стоит отсортировать их по фамилии (будем считать таковой последнее слово ФИО);
количество фильмов, снятых не в США.
Фильмы должны выводиться информативно и красиво, например так:
Terminator 2: Judgment Day (1991-07-03; Action/Sci-Fi) - 137 min — так, чтобы видно
было все интересующие нас поля. Для этого стоит написать свой метод, который выводит массив фильмов.
